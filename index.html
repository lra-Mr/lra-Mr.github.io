博客搭建

​ 由于许久没有使用博客了，导致博客不可用，于是就把它卸载重新搭建，也是相当于巩固一下吧！

话不多说开始吧

​ 搭建博客需要安装以下的东西：

1、下载Git

2、下载node.js

3、安装hexo

下载Git

方式一(官网下载)：https://git-scm.com/download/win(到此页面后，会有一个下载弹框出现)

方式二(csdn下载)：http://download.csdn.net/download/ainuser/9952103

​ 安装git一路向下next就可以了

下载node.js

下载方式一：http://nodejs.org/download/

下载方式二：https://nodejs.org/en/

下载方式三：http://download.csdn.net/download/ainuser/9952074

安装步骤基本一样

安装hexo

首先在磁盘中建立一个hexo的文件夹用于储存文件。

然后在hexo文件夹中右键点击打开git bash here 命令框，

在命令行中依次输入:

​ npm install -g hexo

​ hexo init

​ npm install

​ hexo server

注意，当出现 http://localhost:4000的时候，可以在浏览器中输入，如果出现本地访问页面的话就说明成功了，如果出现404的话那就错误则可以尝试下面的命令

​ npm install hexo-deployer-git –save或者

​ npminstall hexo-server –save

如果出现本地页面的的话就忽略吧

按Ctrl+c退出继续以下的命令

​ hexo clean

​ hexo generate

​ hexo deploy

然后是域名的购买、解析等，就不举例了

当域名有了后在hexo文件夹中的source中建立一个以CNAME命名的文件(切记把后缀去掉）

内容为你购买的域名如：www.xxxxxx.xyz 然后保存关闭
