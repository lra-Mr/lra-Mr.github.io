博客搭建

​由于许久没有使用博客了，导致博客不可用，于是就把它卸载重新搭建，也是相当于巩固一下吧！

话不多说开始吧

​ 搭建博客需要安装以下的东西：

1、下载Git

2、下载node.js

3、安装hexo

下载Git

方式一(官网下载)：https://git-scm.com/download/win(到此页面后，会有一个下载弹框出现)

方式二(csdn下载)：http://download.csdn.net/download/ainuser/9952103

​ 安装git一路向下next就可以了

下载node.js

下载方式一：http://nodejs.org/download/

下载方式二：https://nodejs.org/en/

下载方式三：http://download.csdn.net/download/ainuser/9952074

安装步骤基本一样

安装hexo

首先在磁盘中建立一个hexo的文件夹用于储存文件。

然后在hexo文件夹中右键点击打开git bash here 命令框，

在命令行中依次输入:

​
npm install -g hexo

​ hexo init

​ npm install

检测是否完整安装hexo,在命令行中输入：

​ hexo server

注意，当出现 http://localhost:4000的时候，可以在浏览器中输入，如果出现本地访问页面的话就说明成功了，如果出现404的话那就错误则可以尝试下面的命令

​ npm install hexo-deployer-git –save或者

​ npminstall hexo-server –save

如果出现本地页面的的话就忽略吧

按Ctrl+c退出继续以下的命令

​ hexo clean

​ hexo generate

​ hexo deploy

然后是域名的购买、解析等，就不举例了

当域名有了后在hexo文件夹中的source中建立一个以CNAME命名的文件(切记把后缀去掉）

内容为你购买的域名如：www.xxxxxx.xyz 然后保存关闭

注册账号

首先注册一个GitHub账户:

github的地址为：https://github.com/

进行一系列的操作，也就是注册GitHub账户，注册好之后可以在setting(设置)里面操作。

SSH key配置

在hexo文件夹中右键点击打开git bash here 命令框

可以输入下列的命令，看看是否有SSH key:
cd ~/.ssh

如果提示 No such file or directory 说明你是第一次使用git

那就要生成新的SSH key:
ssh-keygen -t rsa -C “github注册时用的邮箱@qq.com”

然后提示输入密码，这个密码要记住，因为在你以后检查使用。

之后就要添加SSH key到GitHub上

在本机设置SSH Key之后，需要添加到GitHub上，以完成SSH链接的设置。

1、打开本地C:\Documents and Settings\Administrator.ssh\id_rsa.pub文件。此文件里面内容为刚才生成人密钥。如果看不到这个文件，你需要设置显示隐藏文件。准确的复制这个文件的内容，才能保证设置的成功。

2、登陆github系统。点击右上角的 Account Settings—>SSH Public keys —> add another public keys
3、把你本地生成的密钥复制到里面（key文本框中）， 点击 add key 就ok了

完成后，输入下列命令看看设置是否成功：
ssh -T git@github

命令输入后它会提示你输入密码，输入刚的密码就可以了。

完成后继续输入以下命令:

git config –global user.name “你的GitHub名字”

git config –global user.email “你自己的邮箱”

域名绑定

接下来就是域名绑定了:

进入GitHub，点击右上角的加号，点击New repository按钮创建仓库， Respository name 中的username.github.io 的username 一定与前面的Owner 一致，记住你的username下面会用到，然后点击Create repository

创建好后点击settings(设置)下拉到GitHub Pagess，点击Choose a theme,选择一个主题，完成后再次回到这里，可以看到出现了一个新的选项Custom domain，在他的框框里输入你已经解析的域名，勾选https,点击save就可以了

域名绑定完成。

现在可以在浏览器输入你的域名看看什么效果吧
